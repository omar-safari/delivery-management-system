# Delivery Management System 🚚

## 📌 Description

This project is a **Delivery Management System** developed using:

* **Spring Boot** (Backend)
* **React** (Frontend)

The application allows managing deliveries, users, and products through a REST API and a modern web interface.

---

## ⚙️ Technologies Used

* Java + Spring Boot
* React.js
* Maven
* REST API
* MySQL 

---

## 📂 Project Structure

```
delivery-management-system/
│── backend/       # Spring Boot API
│── frontend/      # React App
│── README.md
```

---

## 🚀 How to Run the Project

### 🔹 Backend (Spring Boot)

```bash
cd backend
mvn spring-boot:run
```

Runs on: http://localhost:8080

---

### 🔹 Frontend (React)

```bash
cd frontend
npm install
npm start
```

Runs on: http://localhost:3000
